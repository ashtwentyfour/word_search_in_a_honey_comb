# Honey Comb Word Search

**Compile:**

```
  g++ -Wall -Werror *.cpp
```

**Run:**

```
  ./a.out honeycomb.txt wordlist.txt
```

**'honeycomb.txt' specifications:**

1.  1st line -> Number of layers (N)
2.  2nd line -> Single character
3.  3rd line -> 6 chars
4.  4th line -> 12 chars
5.  Nth line -> No of characters in the (N-1) th line + 6 characters
